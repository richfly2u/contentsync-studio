# ContentSync Studio — 域名命名建議

## 專案定位

ContentSync Studio 是一個 **影片內容處理 SaaS**，核心功能：

| 功能 | 英文關鍵字 | 說明 |
|:---|:---|:---|
| 🎬 影片下載 | `video download` | 貼連結 → 下載 MP4（支援 50+ 平台） |
| 📝 文案提取 | `caption extract` | 影片轉文字（AI 語音辨識） |
| 🎵 影片轉音頻 | `audio extract` | 提取 MP3 |
| 🖼️ 圖片文字辨識 | `OCR` | 圖片中提取文字 |
| 🤖 AI 優化 | `AI optimize` | 清除口頭禪、改寫文案 |
| 📤 多平台發布 | `multi-publish` | 一鍵發布到各平台 |

> 一句話：「**貼連結 → 取內容 → 再發布**」

---

## 命名策略

### 品牌名關鍵字組合

```
[動作/對象] + [後綴]
```

**動作/對象候選：**
- `vid` / `video` — 影片
- `clip` — 片段/剪輯
- `content` — 內容
- `media` — 媒體
- `get` / `fetch` / `pull` — 提取/下載
- `go` / `now` — 快速、即用

**後綴：**
- `.app` — SaaS 感最強，強制 HTTPS（SEO 加分）
- `.io` — 開發者/科技感
- `.cc` — 短、中文圈常見
- `.xyz` — 便宜但品牌感較弱

## 可用域名排名

| 項目 | 說明 |
|:---|:---|
| 字義 | `vid`（影片）+ `go`（開始/進行） |
| 字數 | **5 字母**，最短 |

## 最終建議

```
🥇 vidgo.app  ← 最短、最好記、品牌潛力最大
```

### 為什麼是 vidgo？

1. **5 字母** — 輸入快，不容易打錯
2. **雙音節** — 英文母語者也能輕鬆唸出來
3. **動詞+動詞** — `vid`（影片）+ `go`（執行），暗示「貼上即執行」
4. **無連字號** — 利於口耳相傳
5. `.app` — Google 管理後綴，強制 HTTPS，SEO 友好
